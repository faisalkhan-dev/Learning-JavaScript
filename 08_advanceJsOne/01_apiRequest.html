In Http request using AJAX(Asynchronous JavaScript and XML),it uses 5 states

0 - UNSENT: The XMLHttpRequest object has been created, but the open() method has not been called yet.
1 - OPENED: The open() method has been called, but the send() method has not been called yet. The request is now ready to be sent.
2 - HEADERS_RECEIVED: The send() method has been called, and the response headers and status code have been received. However, the body of the response is not yet available.
3 - LOADING: The response body is being received. The data is being downloaded, but it's not complete yet.
4 - DONE: The operation is complete, and the full response has been received. This is the state where you can access the response data.
